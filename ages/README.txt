The program is on ChessGA.htm

It is recommended it is opened using Firefox.

Description of Files:

	ChessGA: Contains the interface to the GA.
	LICENSE-Blaze: Contains the license of the GA
	LICENSE-garbojs: Contains the license of the original chess AI
	Parser: Creates statistical data from the history acquired from ChessGA
	unsafe_chrome: This opens an instance of Chrome that allow background workers
			to be executed from a local context.
	Paper: This contains the Paper explaining the GA to great detail.
